## 📱 **الشاشات الرئيسية المطلوبة**

### **1. الشاشة الرئيسية (Home Screen)**
**الوظيفة:** عرض المهام اليومية والسريعة
**العناصر المطلوبة:**
- بطاقة ترحيب ذكية مع اسم المستخدم والوقت
- شريط تقدم اليومي
- قائمة المهام مع بطاقات عصرية
- FAB متعدد الوظائف (Speed Dial)
- شريط بحث ذكي مع اقتراحات
- فلاتر سريعة (اليوم، هذا الأسبوع، الأولوية)

### **2. شاشة إضافة/تعديل المهمة**
**الوظيفة:** إنشاء وتحرير المهام
**العناصر المطلوبة:**
- حقل العنوان مع اقتراحات ذكية
- حقل الوصف (اختياري)
- منتقي التاريخ والوقت
- منتقي الأولوية (ألوان متدرجة)
- منتقي الفئة
- زر الحفظ مع تأثيرات بصرية
- اختصار الإدخال الصوتي

### **3. شاشة الإحصائيات (Dashboard)**
**الوظيفة:** عرض الإنتاجية والتحليلات
**العناصر المطلوبة:**
- رسم بياني للإنتاجية الأسبوعية
- بطاقات الإنجازات (المهام المكتملة، الأهداف)
- أفضل وقت للعمل (تحليل ذكي)
- شارات الإنجاز
- تقارير شهرية

### **4. شاشة الإعدادات**
**الوظيفة:** تخصيص التطبيق
**العناصر المطلوبة:**
- اختيار الثيم (فاتح/داكن/تلقائي)
- إعدادات الإشعارات
- إعدادات الإدخال الصوتي
- لغة الواجهة
- النسخ الاحتياطي

---

## ✨ **المميزات التصميمية المطلوبة**

### **1. Neo-morphism Cards**
- بطاقات ثلاثية الأبعاد مع ظلال ناعمة
- تأثيرات الضغط (Press states)
- حواف مدورة بشكل حديث

### **2. Micro-interactions**
- Haptic feedback لكل إجراء
- Spring animations سلسة
- Particle effects عند إكمال المهام
- Loading states جميلة

### **3. Smart FAB (Speed Dial)**
- زر رئيسي مع قائمة منبثقة
- أيقونات واضحة مع تسميات
- حركات سلسة للفتح والغلق
- اختصارات سريعة (إضافة، صوتي، صورة، مجدول)

### **4. Dark Mode Plus**
- 3 خيارات للوضع الليلي
- تدرجات لونية مختلفة
- تبديل تلقائي حسب الوقت

### **5. Arabic Typography**
- خط NotoSansArabic كخط أساسي
- أحجام خطوط مناسبة للعربية
- مسافات مناسبة بين الكلمات
- دعم كامل للـ RTL
